# 🌤️ Weather App

A modern Android Weather application built with Jetpack Compose, MVVM architecture, and Hilt dependency injection.

## Features
- City search
- Real-time weather (OpenWeather API)
- Forecast display
- Dynamic UI based on weather
- Clean MVVM architecture

## Tech Stack
- Kotlin
- Jetpack Compose
- Hilt
- Retrofit
- Coroutines/Flow
- Material 3

## Architecture
UI → ViewModel → Repository → API

## Setup
git clone https://github.com/zaydkassimi/WeatherApp.git

Add in local.properties:
OPEN_WEATHER_API_KEY=your_api_key_here
